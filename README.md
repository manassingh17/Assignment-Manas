# Gushwork Assignment — Manas Singh

A responsive product landing page for Euroflux (HDPE Pipes & Fittings), built with vanilla HTML, CSS, and JavaScript.

**Live:** [gushwork-assignment-bay.vercel.app](https://gushwork-assignment-bay.vercel.app/)

## Features

- Sticky header that activates after scrolling past the hero
- Image carousel with hover-based zoom
- Tabbed manufacturing process section
- Infinite-scroll carousels for applications and testimonials
- FAQ accordion
- Modal dialogs for brochure download and callback requests
- Fully responsive across desktop, tablet, and mobile

## Tech Stack

- HTML5 (semantic markup, ARIA attributes)
- CSS3 (custom properties, flexbox, grid, media queries)
- Vanilla JavaScript (no frameworks or libraries)
